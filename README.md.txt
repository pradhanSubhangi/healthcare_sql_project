# 🩸 Personalized Healthcare Recommendations — SQL Analysis

## 📌 Project Overview
Data analysis on blood test results to identify patterns and 
insights across different diseases using MySQL.

## 🛠️ Tools Used
- MySQL 8.0
- MySQL Workbench
- Git and GitHub

## 📊 Analysis Performed
1. Missing Values Check
2. Disease Distribution Analysis
3. Average Blood Values by Disease
4. High Risk Patient Identification
5. Heart Rate Analysis by Disease
6. Cholesterol Analysis (LDL, HDL, Triglycerides)
7. Blood Pressure Analysis
8. Liver Function Analysis (ALT & AST)
9. Overall Summary Statistics

## 🔍 Key Findings
- Dataset contains 2,351 patient records
- Multiple diseases analyzed including Diabetes, Heart Disease etc.
- High risk patients identified based on Glucose and BMI levels
- Blood pressure, cholesterol and liver function patterns found

## 📁 Dataset
Source: Kaggle — Blood Sample Dataset
File: blood-sample-dataset-balanced.csv
